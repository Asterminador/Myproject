O jogo comeca pelo elemento vida, depois vai para o elemento terra, elemento agua e finalmente o elemento ar. O grupo que responde é o grupo do elemento ativo. O grupo pode rolar os dados para definir as suas ações.

Caso o dado sorteie o numero 1 o grupo deve apontar um dano referente ao seu elemento que esta afetando negativamente o ambiente da Yby

Caso o dado sorteie o numero 2 o grupo deve apontar um dano futuro (previsto baseando-se no estudo)

Caso o dado sorteie o numero 3 o grupo pode contra atacar o vilão. Em posse da arma secreta, ao oferecer uma solução pra alguns dos problemas ja listados anteriormente(o dado nunca começa no 3) o grupo atinge o vilão bem no coração, ganhando uma pontuação especial para Yby
